This purpose of this project is :
- The NodeMCU captures the values of the temperature from DHT11.
- Post them with their TimeStamp to MySQL Local Database.

-The form of the SQL Database:
	temps
------------------------
temp_c   |  temp_date
------------------------
float	 | timestamp
	 |

- Display the captured data on in a line chart (http://localhost/chartjs/linegraph.html)
